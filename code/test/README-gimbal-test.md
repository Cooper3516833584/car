# L150 C10B A2/A3 云台临时测试

当前驱动板固件没有 A2/A3 串口命令，所以先在 Keil 工程中应用
`c10b_a2_a3_gimbal.patch`、重新编译并按 C10B 的下载流程刷入；刷写前应断开
云台的机械阻挡，并保留可恢复的原始固件。

刷写后，在 ROCK 5A 上以管理员权限运行：

```bash
sudo python3 /home/radxa/gimbal_host_test.py --a2 1650 --a3 1450
```

该测试将连续发送 1.5 秒的 20 Hz 指令。A2 是水平轴、A3 是竖直轴；两个值均为
微秒脉宽，已限制在 900-2100 us。命令帧中的底盘线速度和角速度始终为零。
